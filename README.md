📊 E-Commerce Sales Analysis Dashboard

🚀 Project Overview

This project analyzes e-commerce sales data using SQL and Power BI to generate business insights.

🛠️ Tools Used

- SQL (Data Analysis)
- Power BI (Dashboard)
- Excel (Dataset)

📌 Key Features

- Total Sales, Profit, Orders KPIs
- Monthly Sales Trend
- Top 10 Sub-Categories
- Sales by Category
- Region-wise Sales & Profit

📂 Project Structure

- "data/" → Dataset
- "sql/" → SQL Queries
- "dashboard/" → Power BI file
- "images/" → Screenshots

📊 Insights

- Clothing category generates highest sales
- West region has lowest profit
- Sales trend increases over time

🧠 Learnings

- Data cleaning using SQL
- Aggregations and grouping
- Data visualization best practices
