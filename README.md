📊 E-Commerce Sales Analysis Dashboard

🚀 Project Overview

This project analyzes e-commerce sales data using SQL and Power BI to generate meaningful business insights.
It includes data cleaning, aggregation, and interactive dashboard creation.

---

🛠️ Tools Used

- SQL (Data Analysis)
- Power BI (Dashboard)
- Excel / CSV (Dataset)

---

⭐ Project Highlights

- End-to-end data analysis project
- Real-world business insights
- Interactive Power BI dashboard

---

📊 Key Features

- Total Sales, Profit, Orders KPIs
- Monthly Sales Trend
- Top Sub-Categories by Sales
- Category-wise Performance
- Region-wise Sales & Profit

---


📂 Project Structure

- "data/" → Dataset file
- "sql/" → SQL queries
- "images/" → Dashboard screenshot
- "dashboard/" → Power BI file

---

📊 Business Insights

- Clothing category contributes highest sales (~35%)
- West region has lowest profit
- Sales trend peaks during mid-year
- Some sub-categories generate high sales but low profit

---

🧠 Learnings

- SQL aggregations and grouping
- Data cleaning techniques
- Power BI dashboard creation
- Business insight generation
